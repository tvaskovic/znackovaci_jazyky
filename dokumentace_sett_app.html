Úvod

Úkolem v této práci bylo vytvoření XML dokumentu, XSD schématu, XSL transformace do JSON a dokumentace na předmět Značkovací jazyky na téma nastavení existující/fiktivní aplikace.

Metodika

Celá tato práce byla vytvořena v textovém editoru Sublime Text 2 (možné stáhnot zde Sublimetext), kdy byl nejprve vytvořen XML dokument na nastavení fiktivní aplikace, v kterém byly vymyšleny vlastní elementy a atributy do kterych byla ukládány informace této aplikace. Poté bylo vytvořeno XML Schéma za pomoci online tranformátoru dostupném zde Freeformater a célé XML bylo přetransformováno do XSD souboru. Následovalo vytvoření XSL dokumentu transformovaného do JSON. Tato transformace proběhla vložením XML souboru do online transformátoru dostupném zde Convertjson.com. Po transformaci XML do XSL byl soubor ručně dotvořen do JSON kódu.

Popis transformace

JSON transformace

Pro transformaci do JSON jsem použil online nástroj Convertjson.com. Po té, co jsem vložil XML kód z mého projektu, musel jsem do dalšího okna začít psát transformaci XSL. Výsledkem byl JSON kód.

Závěr

Práce na tomto projektu byla velmi přínosná a naučíl jsem se díky ní mnoho nového ze světa značkovacích jaziků. Nejnáročnější z celé práce byla transformace do JSON. Práce vznikala za použití textového editoru Sublime Text 2, který zvýrazňuje elemty a atributy čimž se stáva zdroják přehlednější.
